# 🤖 WhatsApp Bot + Geophysics + n8n Automation Project

This is an integrated project combining a **WhatsApp Bot**, **Geophysics data analysis**, and **Python + MATLAB automation**, all connected via **n8n workflows**.

> 🔧 Built and maintained by [Ivan Hutabarat](https://github.com/Ivanhutabarat)

---

## 📱 WhatsApp Bot Features

- 📥 Auto-replies to incoming messages
- 📊 Integrates with data analysis results from MATLAB/Jupyter
- 🧠 Responds with real-time geophysical data summaries
- 🔄 Fully automatable using n8n workflows

---

## 🧪 Geophysics Data & Analysis

- 📎 MATLAB scripts for seismic data processing
- 📓 Jupyter Notebooks for waveform visualization and analysis
- 🛰️ Earthquake event detection (ObsPy-based)
- 📁 Integration with Python for automatic processing pipelines

---

## 🔁 n8n Workflow Integration

- 🧩 Orchestrates all tasks: message parsing, script execution, and data returns
- 🔄 Automates replies with geophysical summaries or charts
- 💬 Real-time messaging with dynamic content from Python/MATLAB results

## 🚀 How to Run (from phone or PC)

1. Clone or download this repo.
2. Install requirements:  
   ```bash
   pip install -r requirements.txt

3. Make sure you have MATLAB installed or use octave for open-source.


4. Set up your n8n instance and import the JSON workflow file.


5. Run main.py to launch the bot!
