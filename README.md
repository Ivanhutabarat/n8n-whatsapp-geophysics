Geophysics WhatsApp Bot & Automation with n8n

Welcome to my all-in-one project that combines WhatsApp bot development, geophysical data analysis, Python automation, and workflow orchestration using n8n.

🔥 Overview

This repository integrates various technologies to demonstrate my multidisciplinary skills in:

🤖 Building a WhatsApp Bot using Python

🌍 Performing Geophysical Data Analysis using MATLAB and Jupyter

⚙️ Automating data tasks with Python scripts

🔁 Managing everything with an n8n Workflow

🚀 Quick Start

# Clone the repository
$ git clone https://github.com/Ivanhutabarat/geophysics-whatsapp-n8n

# Set up Python environment
$ cd geophysics-whatsapp-n8n
$ pip install -r requirements.txt

# Run the WhatsApp bot
$ python whatsapp-bot/bot.py
